# LOST

## Description

A simple but incredible Platformer Game made by TDM Studios! 
This is a Project for Videogames Development subject in CITM University, hope you like it!

## Key Features

 - Load & Save Feature
 - Map Navigation
 
## Controls

 - WASD - Movement controls.
 - Space Bar - Jump.
