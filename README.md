# LOST

## Description

A simple yet amazing platform game created by TDM Studios! 
This project is created according to the Videogame Development subject at CITM university. 
We hope you like it!

## Key Features

 - Load & Save Feature
 - Map Navigation
 
## Controls

 - WASD - Movement controls.
 - Space Bar - Jump.
